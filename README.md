# react-starter-kit
a react program starter kit

## features
* eject create-react-app
* add antd react-router-dom less
* enable CSS Modules
* extend meta

## usage
* `npm i`
* edit `name` in `package.json` and `public/manifest.json`
* edit `title` in `public/index.html`
